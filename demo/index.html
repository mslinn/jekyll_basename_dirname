Basename

Current page

This example returns the basename of this page, which has the url {{page['url']}}. 

{% raw %}{{ page['url'] | basename }}{% endraw %}

Renders as:

{{ page['url'] | basename }}

Arbitrary Path

This example returns the basename of an arbitrary path, {{blah}}. 

{% raw %}{{ blah | basename }}{% endraw %}

Renders as:

{{ blah | basename }}

WBR

Current page

This example returns the full path of this page, such that it wraps, which has the url {{page['url']}}. 

{% raw %}{{ page['url'] | wbr }}{% endraw %}

Renders as:

{{ page['url'] | wbr }}

Renders as (escaped):

{{ page['url'] | wbr | escape }}

This example returns the full path of an arbitrary path, such that it wraps, {{blah}}.

Renders as:

Renders as (escaped):

{{ blah | wbr | escape }}
